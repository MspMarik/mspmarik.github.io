Dating Ducks



Dating Ducks GitHub

The semester long project for my Web Programming II class was to make a website of my team's choosing using the technologies we learned in class as well as outside technologies of our choosing. My team chose the MERN tech stack .

My group created a dating website exclusively for Stevens students. During the signup process, a new user fills in information about themselves and is then matched with other users based on that information. In a similar fashion to Tinder or other dating sites, users can like or dislike a user presented to them. If two users like each other, they are matched. Users can join public chat rooms to talk with their matches.

Technologies used:

  • Node.js
  • React
  • Bootstrap 5
  • MongoDB
  • CSS
  • Express
  • Firebase (for user authentication)
  • ImageMagick
  • Socket.io
  • Docker
